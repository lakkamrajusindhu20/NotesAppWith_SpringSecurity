# Notes Management System with Spring Security

A secure **Notes Management Web Application** built with **Spring Boot**, **Spring Security (JWT)**, **MySQL**, and **JSP** views.

This project allows users to **register**, **log in**, and **manage their personal notes** with authentication using **JWT (JSON Web Tokens)**.  
It demonstrates a full-stack backend with security, persistence, and REST APIs.

---

##  Features

✔ User Registration (Signup)  
✔ User Login with JWT Authentication  
✔ Create, Read, Update, Delete (CRUD) Notes  
✔ Secure APIs using JWT  
✔ Role-based access support (can be extended)  
✔ Uses MySQL database for persistence  
✔ JSP based frontend views (optional UI)

## Technology Stack

| Technology | Purpose |
|------------|---------|
| **Java** | Core language |
| **Spring Boot** | Application framework |
| **Spring Security** | Authentication & Authorization |
| **JWT (JSON Web Tokens)** | Token-based auth |
| **MySQL** | Database |
| **JPA / Hibernate** | ORM for database access |
| **Maven** | Build tool |
| **JSP** | Frontend views |
